:::info[no-header]
Provides a detailed specification of the pitch location where a shot was taken, breaking down the area around the goal into precise zones for granular analysis.
:::

---

## Enum Values

| Value | Description |
|---|---|
| `UNSPECIFIED` | Represents a NULL or UNKNOWN value. |
| `BOX_SMALL_RIGHT` | The action was taken in the right side of the six-yard box. |
| `BOX_SMALL_CENTER` | The action was taken in the center of the six-yard box. |
| `BOX_SMALL_LEFT` | The action was taken in the left side of the six-yard box. |
| `BOX_RIGHT` | The action was taken in the right side of the penalty area. |
| `BOX_CENTER` | The action was taken in the center of the penalty area. |
| `BOX_LEFT` | The action was taken in the left side of the penalty area. |
| `BOX_DEEP_RIGHT` | The action was taken deep in the right side of the penalty area. |
| `BOX_DEEP_LEFT` | The action was taken deep in the left side of the penalty area. |
| `BOX_OUT_RIGHT` | The action was taken outside the penalty area on the right side. |
| `BOX_OUT_CENTER` | The action was taken outside the penalty area in a central position. |
| `BOX_OUT_LEFT` | The action was taken outside the penalty area on the left side. |
| `BOX_OUT_DEEP_RIGHT` | The action was taken far outside the penalty area on the right side. |
| `BOX_OUT_DEEP_LEFT` | The action was taken far outside the penalty area on the left side. |
| `PLUS_35M_RIGHT` | The action was taken more than 35 meters from goal on the right side. |
| `PLUS_35M_CENTER` | The action was taken more than 35 meters from goal in a central position. |
| `PLUS_35M_LEFT` | The action was taken more than 35 meters from goal on the left side. |
