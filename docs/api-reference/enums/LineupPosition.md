:::info[no-header]
Specifies the exact role and typical location of a player on the field within the team's starting lineup.
:::

---

## Enum Values

| Value | Description |
|---|---|
| `GOALKEEPER` | The goalkeeper, the only player allowed to handle the ball within their own penalty area. |
| `WINGBACK` | General wide defender/wingback position |
| `WINGBACK_LEFT` | Left-sided wingback. |
| `WINGBACK_RIGHT` | Right-sided wingback. |
| `FULLBACK` | General wide defender/fullback position. |
| `FULLBACK_LEFT` | Left-sided fullback. |
| `FULLBACK_RIGHT` | Right-sided fullback. |
| `CENTREBACK` | General central defender position. |
| `CENTREBACK_LEFT` | Left-sided central defender (LCB). |
| `CENTREBACK_RIGHT` | Right-sided central defender (RCB). |
| `MIDFIELDER` | General central midfield position. |
| `MIDFIELDER_LEFT` | Wide midfield position on the left flank (LM). |
| `MIDFIELDER_CENTRE` | Standard central midfield position (CM). |
| `MIDFIELDER_RIGHT` | Wide midfield position on the right flank (RM). |
| `MIDFIELDER_CENTRE_LEFT` | Central midfielder operating slightly to the left side. |
| `MIDFIELDER_CENTRE_RIGHT` | Central midfielder operating slightly to the right side. |
| `DEFENSIVE_MIDFIELDER` | General midfield position primarily focused on defensive tasks. |
| `DEFENSIVE_MIDFIELDER_LEFT` | Defensive midfielder operating on the left side. |
| `DEFENSIVE_MIDFIELDER_RIGHT` | Defensive midfielder operating on the right side. |
| `DEFENSIVE_MIDFIELDER_CENTRE` | The primary defensive central midfield position (CDM) often referred to as the 'Number 6'. |
| `DEFENSIVE_MIDFIELDER_CENTRE_LEFT` | Central defensive midfielder operating slightly to the left. |
| `DEFENSIVE_MIDFIELDER_CENTRE_RIGHT` | Central defensive midfielder operating slightly to the right. |
| `ATTACKING_MIDFIELDER` | General midfield position primarily focused on attacking and playmaking. |
| `ATTACKING_MIDFIELDER_LEFT` | Attacking midfielder operating on the left side. |
| `ATTACKING_MIDFIELDER_RIGHT` | Attacking midfielder operating on the right side. |
| `ATTACKING_MIDFIELDER_CENTRE` | The central attacking midfield position (ACM), often referred to as the 'Number 10'. |
| `ATTACKING_MIDFIELDER_CENTRE_LEFT` | Central attacking midfielder operating slightly to the left. |
| `ATTACKING_MIDFIELDER_CENTRE_RIGHT` | Central attacking midfielder operating slightly to the right. |
| `STRIKER` | General forward position. |
| `STRIKER_LEFT` | Forward operating on the left side. |
| `STRIKER_RIGHT` | Forward operating on the right side. |
| `STRIKER_CENTRE` | The central forward position, often referred to as the 'Number 9'. |
| `STRIKER_CENTRE_LEFT` | Central striker operating slightly to the left. |
| `STRIKER_CENTRE_RIGHT` | Central striker operating slightly to the right. |
| `WINGER` | General wide attacking position, typically a fast player providing crosses and goals. |
| `WINGER_LEFT` | Left-sided winger (LW). |
| `WINGER_RIGHT` | Right-sided winger (RW). |
