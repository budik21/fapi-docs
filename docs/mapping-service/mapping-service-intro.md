## What is the Mapping Service?

The Mapping Service allows you to resolve **StatsPerform IDs** to their corresponding **Snowflake IDs** in the Livesport Football Database.

If your system already uses StatsPerform IDs, you do not need to search for entities by name — you can use this service to obtain Snowflake IDs directly and start querying the Football API immediately.

## Supported ID Types

| Source System  | Supported |
|----------------|-----------|
| StatsPerform   | Yes       |
| Others         | No        |
