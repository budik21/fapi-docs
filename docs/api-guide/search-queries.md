## What are Search Queries?

Search Queries are a set of Football API queries that allow you to find entities in the Livesport Football Database **by name or other attributes**, rather than by ID.

They are the recommended starting point if you do not have StatsPerform IDs and need to obtain Snowflake IDs before querying the Football API for match data, statistics, or other information.

## When to Use Search Queries

Use Search Queries when:
- You are integrating the Football API for the first time and have no existing compatible IDs.
- You need to look up a competition, team, player, or match by name.
