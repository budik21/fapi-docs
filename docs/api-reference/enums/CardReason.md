:::info[no-header]
Indicates the specific reason why the referee issued a card to a player or coach, either during or after the match.
:::

---

## Enum Values

| Value | Description |
|---|---|
| `UNSPECIFIED` | Represents a NULL or UNKNOWN value |
| `SERIOUS_FOUL` | A foul with high risk of injury, using excessive force or brutality, like a dangerous tackle (direct red card) |
| `HANDBALL` | Intentional use of the hand or arm to play the ball |
| `REFEREE_ABUSE` | Insulting or attacking the referee |
| `ARGUMENT` | Arguing with the referee or other players |
| `VIOLENT_CONDUCT` | Physical violence not related to playing the ball |
| `TIME_WASTING` | Deliberately delaying the game to gain an advantage |
| `CROWD_INTERACTION` | Unauthorized interaction with spectators |
| `SIMULATION` | Faking a foul to deceive the referee such as diving etc. |
| `PERSISTENT_INFRINGEMENT` | Repeatedly breaking the rules |
| `FOUL_AND_ABUSE` | A foul combined with offensive behavior |
| `ENCROACHMENT` | Moving too close to the ball at a free kick, failing to respect the required distance – for example, a player rushing out of the wall to challenge the kick before it is taken. |
| `PITCH_LEAVING` | Leaving the field without the referee’s permission |
| `PITCH_ENTERING` | Entering the field without the referee’s permission |
| `SPITTING` | Spitting at an opponent or any other person on the field |
| `DISSENT` | Showing disagreement with the referee, often verbally or through gestures |
| `OFF_THE_BALL_FOUL` | A foul committed away from the ball |
| `UNSPORTSMANLIKE_BEHAVIOR` | Any behavior that goes against fair play |
| `RECKLESS_FOUL` | A foul committed in a reckless manner, ignoring the risk to the opponent (common yellow card) |
| `TACTICAL_FOUL` | A deliberate foul committed to stop or slow down an opponent’s promising attack, usually resulting in a yellow card. |
| `TACTICAL_FOUL_LAST_PLAYER` | A deliberate foul committed to stop or slow down an opponent’s promising attack, committed by a last man in the defence (usually resulting in a red card) |
| `NOT_RETREATING` | The player does not move back the appropriate distance e.g. player in a wall does not retreat |
